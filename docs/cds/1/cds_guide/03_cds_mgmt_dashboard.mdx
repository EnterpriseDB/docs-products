---
title: "The Management Dashboard Tab"
---

Select an AWS region from the options displayed on the `Dashboard` tab to spin up a new console.

![*The Dashboard tab.*](images/cds_mgt_dashboard.png)

CDS will open a new browser tab in the selected region.

![*The Clusters tab.*](images/cds_mgmt_console_cluster_tab.png)

To provision a database cluster, select the `New Cluster` icon in the left panel of the `Clusters` tab.

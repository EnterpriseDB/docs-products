---
title: Supported Platforms
---

The EDB Ark management console runs on the following browser versions
(or newer):

  - Mozilla Firefox 18
  - Mozilla Firefox 17 ESR, 24 ESR, 31 ESR
  - Internet Explorer 8
  - Safari 6
  - Opera 16
  - Google Chrome 23

EDB Ark provisions cluster instances on the following 64-bit Linux
systems:

  - RHEL 7.x
  - CentOS 7.x
